Title

The HTML head element is a container for the following elements: title, style, meta, link, script, and base. The head element is a cointainer for metadata. This data is not displayed on the webpage. Metadata typically define the document title, character set, styles, scripts, and other meta information.

The Viewport

The viewport is the user's visible area of a web page. It varies with the device - it will be smaller on a mobile phone than on a computer screen.
